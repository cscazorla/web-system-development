Final Project

  • Build a fully functional full-stack web application using the technologies covered in the course.
  • The project should allow user authentication, CRUD operations, API communication, and deployment with CI/CD.

Requirements

  • Frontend: Use Next.js with TypeScript. Implement user authentication (JWT). Create a dashboard with CRUD operations. Fetch & display data from the backend API.
  • Backend: Set up web server. Implement PostgreSQL database. Provide RESTful API endpoints for CRUD operations. Secure routes with authentication. Implement unit & integration tests.
  • Deployment & DevOps: Containerize the backend using Docker. Deploy the backend and the frontend. Set up CI/CD with GitHub Actions. Configure monitoring.

Deliverables

  • Github repository with the code of the project.
  • Live deployed version
  • API Documentation (Swagget/Postman)
  • CI/CD pipeline using GitHub Actions
